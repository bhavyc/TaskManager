# TaskApp - Frontend Developer Intern Assignment

## Overview
**TaskApp** is a full-stack web application built as part of a Frontend Developer Intern assignment.  
The app allows users to **register/login**, manage their tasks (CRUD), and **search/filter tasks**.  
The frontend is built with **React.js + Bootstrap**, and the backend is built with **Node.js + Express + MongoDB**, with **JWT-based authentication**.

---

## Features

### Authentication
- User Registration
- User Login
- JWT-based authentication
- Protected dashboard routes

### Task Management (CRUD)
- Create new tasks
- Read/view all tasks
- Update/edit tasks
- Delete tasks
